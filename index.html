03. Mes Projets

IA branding engine

Votre identité de marque, réinventée par l'IA. Un moteur intelligent basé sur Gemini qui fusionne stratégie et design pour générer vos noms, palettes de couleurs et polices en quelques secondes.

CryptSafe - Interface d'Authentification

Développement d'un portail de connexion sécurisé servant de surcouche graphique pour un outil de chiffrement de fichiers. L'interface, intégrant les bonnes pratiques OWASP contre les failles d'authentification, permet de déverrouiller un espace de stockage local chiffré en AES-256.

NetWatch - Tableau de Bord de Surveillance Réseau & SSI

Conception d'un dashboard de monitoring s'inspirant des architectures de SIEM. Il centralise visuellement les alertes de sécurité, l'état d'activité des pare-feux, et le contrôle des utilisateurs connectés au système informatique. La mise en page utilise CSS Grid pour organiser efficacement les différents modules d'analyse.

SecurLib - Refonte d'une Bibliothèque Numérique

Projet d'intégration front-end d'une landing page moderne pour une bibliothèque en ligne.

Purem Oda - Maquette E-Commerce

Création d'une interface e-commerce minimaliste et haut de gamme mettant l'accent sur le positionnement asymétrique des cartes de produits.

Zonixx App - Interface de Suivi de Performance

Intégration d'une interface web responsive orientée "Mobile First" pour une application de coaching sportif. Le projet étudie l'implémentation de protocoles d'authentification forte (2FA) et la sécurisation des API Rest qui transmettent les données de santé sensibles des utilisateurs vers les bases de données.
